# Portfolio React (Vite)

Sebuah template portofolio sederhana menggunakan React dan Vite.

Quickstart (PowerShell):

```powershell
# 1) Install dependencies
npm install

# 2) Jalankan dev server
npm run dev

# 3) Buka browser pada alamat yang ditampilkan (default http://localhost:5173)
```

File penting:
- `index.html` — entry
- `src/main.jsx` — entry React
- `src/App.jsx` — layout utama
- `src/components/*` — komponen halaman
- `src/components/*` — komponen halaman

Silakan ganti teks dan proyek contoh dengan informasi Anda sendiri. CV (PDF) disertakan di `public/Resume.pdf` — Anda dapat mengunduhnya dari situs menggunakan tombol "Download CV".
 
 Branding: situs ini disiapkan untuk menampilkan Anda sebagai "System Analyst & Tester" — judul halaman, hero, dan link CV sudah diperbarui untuk mencerminkan peran tersebut. Jika Anda ingin variasi branding (mis. "System Analyst / QA Engineer" atau menambahkan sertifikasi), beri tahu saya dan saya akan sesuaikan.
